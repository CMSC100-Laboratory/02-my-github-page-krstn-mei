Steps on how to create GitHub page:  
- Clone the remote GitHub repository to a folder of your choosing.  
- Configure Git in your local repository (set name and email).  
- Push your code to the GitHub repository.  
- Change the visibility of your repository to public.  
- Select which brand you will use to host your website.  

Takeaways:  
- Utilizing flex for moving containers in the page.  
- Using CSS to style and add effects to the web page.
- Efficiency of GitHub pages in hosting static websites.
